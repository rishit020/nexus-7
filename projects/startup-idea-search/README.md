## Criteria (Scoring Grid)

When filtering week 1 → 3 finalists, score each problem 1-5 on:

1. **Market size** — could this be a $10M ARR company?
2. **My fit** — can I build and sell this better than most?
3. **Passion** — will I still want this in 12 months when it's hard?
4. **Moat** — what defensibility could I build (data, distribution, speed)?
5. **Speed to MVP** — can I have something real in 4 weeks?
