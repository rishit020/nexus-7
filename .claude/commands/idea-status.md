# /idea-status

## What you do

1. Read `projects/startup-idea-search/README.md`
2. Check today vs. sprint window (Apr 21 – May 5, 2026)
3. Output:

```
## STARTUP IDEA SPRINT

Days until decision (2026-05-05): X

### Ideas researched
- [list from tracker]

### Ideas validated (talked to real person)
- [list]

### Current frontrunner
- [name or "none yet"]

### Verdict
[On track to decide May 5? Yes/No, one sentence why.]
```

## Rules

- If <5 days to decision and 0 ideas validated → call it out hard. This is a blown sprint.
- If ideas are all CV-flavored → flag that. Criteria says non-CV.
- Don't pick the idea for him. Just reflect the data.
